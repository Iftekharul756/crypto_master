# Flutter Crypto APP
Complete Flutter Application with Provider + Dio for API REST. (Binance API) [University Simple project]
# Features
1. API REST (Binance API, newsapi)
2. Candle Graph View (Hour, Day, Week, etc)
3. Wallet Processes
4. Cryptocurrency Trade
5. Onboarding Pages with Lottie Animations
6. Crypto Related News
7. Popular Cryptocurrencies

# Stack
1. Flutter 3.3.2
2. Dart 2.18.1
3. Provider
4. Dio
5. Lottie
6. syncfusion_flutter_charts
